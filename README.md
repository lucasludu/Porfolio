A professional, high-impact technical portfolio built with modern Web technologies. It features a clean, responsive design, smooth scroll animations, and a fully functional English/Spanish bilingual toggle switch.

## ✨ Features

- **Bilingual Support**: Instant toggle between Spanish (ES) and English (EN) across the entire application without reloading the page.
- **Dynamic CV Download**: Automatically links to either the Spanish or English version of the CV depending on the active language.
- **Beautiful Animations**: Powered by `framer-motion` for fluid page load transitions, scroll reveals, and micro-interactions on hover.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing domains, utilizing Tailwind CSS utility classes.
- **Glassmorphism UI**: Modern aesthetic utilizing backdrop blurring and elegant, low-opacity card backgrounds.
- **"Show More" Functionality**: Intelligently limits the initial display size of project and experience lists, offering a clean, uncluttered UX while allowing deep dives via expandable sections.

## 🛠️ Technology Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: Designed for GitHub Pages

## 📐 Project Structure

The codebase is organized following best practices for a scalable React application:

```
src/
├── components/          # Reusable UI components
│   ├── Hero.jsx         # Main introduction section
│   ├── Navbar.jsx       # Navigation and language toggle
│   ├── ProjectGrid.jsx  # Expandable portfolio gallery
│   ├── TechStack.jsx    # Categorized skill badges
│   └── Timeline.jsx     # Expandable work experience timeline
├── translations.js      # Centralized ES/EN dictionary
├── App.jsx              # Root component & page assembly
├── index.css            # Global Tailwind & custom base styles
└── main.jsx             # React DOM entry point
```

## 🚀 Getting Started Locally

To run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lucasludu/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
